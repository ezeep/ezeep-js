This example demonstrates the simplest integration of the ezeep.js component with a predefined print button.

How it Works

  • Displays a predefined print button
  • Opens the print dialog when clicked
  • File is provided via a URL (fileurl)
  • Custom file names and types can be specified

Attributes Used

  • clientid: Your registered Client-ID
  • redirecturi: Your registered Redirect URI
  • trigger="button": Renders a print button
  • fileurl: URL to the file to be printed
  • filename: Name of the file to be printed
  • filetype: File type (e.g. "pdf")
