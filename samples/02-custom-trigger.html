This example shows how you can use your own HTML elements as triggers for the print dialog. You have full control over the design and behavior of the buttons.

How it Works

  • The ezeep component is initialized with trigger="custom"
  • You create your own HTML elements (buttons, links, etc.)
  • With JavaScript, you call ezpPrinting.open() to open the print dialog
  • Complete design freedom for your own custom styling

Attributes Used

  • clientid: Your registered Client-ID
  • redirecturi: Your registered Redirect URI
  • trigger="custom": No predefined element is rendered
  • fileurl: URL to the file to be printed
  • filename: Name of the file to be printed
  • filetype: File type (e.g. "pdf")
