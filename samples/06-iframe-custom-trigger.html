  • Custom button design - total freedom!

How it Works

  • Create EzeepEmbed instance with trigger="custom"
  • Design your own buttons with any style you want
  • Call printer.print() when button is clicked
  • Print dialog appears in a modal overlay
  • Complete design freedom for your branding

Configuration Used

  • trigger: "custom": No default UI, you control everything
  • theme: "pink": UI color theme for the print dialog
  • appearance: "system": Matches system light/dark mode
