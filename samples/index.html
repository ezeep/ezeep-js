Welcome!

This collection showcases various ways to integrate ezeep.js into your web application. Each example is fully functional and demonstrates a specific use case.

Before you start:

  • Make sure you have an ezeep Blue account (sign up here)
  • Install the ezeep Blue Connector on a device with printer access
  • Examples are pre-configured with a test Client-ID for localhost:8443
  • Serve these files over HTTPS (required for OAuth)
